Gerador de Ícones para PWA - OCTO

Este guia descreve os ícones necessários para a funcionalidade PWA do projeto OCTO. Todos os ícones devem ser colocados na pasta public/icons/.

Nota: É recomendável usar uma única imagem de origem de alta resolução (pelo menos 1024x1024 pixels) e redimensioná-la para os diversos tamanhos necessários, mantendo a consistência visual.

Ícones Necessários

Nome do Arquivo Tamanho Propósito
android-chrome-192x192.png 192x192 Ícone padrão para Android/Chrome
android-chrome-512x512.png 512x512 Ícone de alta resolução para Android/Chrome
apple-touch-icon.png 180x180 Ícone para dispositivos iOS
favicon-32x32.png 32x32 Favicon para navegadores

Usando Ferramentas Online

Você pode usar as seguintes ferramentas para gerar os ícones necessários:

  • Real Favicon Generator - Gera todos os ícones necessários a partir de uma única imagem
  • Maskable.app - Para criar ícones maskable específicos para PWA

Guia de Implementação

  1. Prepare uma imagem de alta resolução com o logotipo OCTO (preferencialmente PNG com fundo transparente)
  2. Use as ferramentas acima para gerar os ícones nos tamanhos corretos
  3. Coloque todos os ícones gerados na pasta public/icons/
  4. Verifique se os caminhos no manifest.webmanifest, vite.config.ts e index.html estão corretos
  5. Teste a aplicação como PWA em diferentes dispositivos
Importante: Para o ícone maskable, certifique-se de que o elemento principal (logotipo) esteja dentro da "zona segura" (área central que ocupa aproximadamente 80% do ícone), pois diferentes dispositivos podem recortar as bordas.
